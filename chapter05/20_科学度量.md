# 20 科学度量：构建 Benchmark 自动化评估脚本，科学量化 Harness 引擎性能

## 🎯 本节目标

设计并实现 Agent Harness 的自动化 Benchmark 评估体系，用数据驱动 Harness 的迭代优化。

---

## 1. 为什么要 Benchmark？

没有 Benchmark，优化就是"凭感觉"：

```
修改 1：改了 System Prompt ← 感觉变好了？
修改 2：换了 Context Compaction 策略 ← 好像快了一点？
修改 3：调整了 max_turns ← 不确定有没有影响...
```

有了 Benchmark：
```
修改前：任务完成率 72%，平均耗时 45s，平均成本 $0.012
修改后：任务完成率 85%，平均耗时 38s，平均成本 $0.009
→ 明确的提升数据，可重复验证
```

## 3. Benchmark 运行器

```python
# benchmark/runner.py

import asyncio
import time
import json
import statistics
from dataclasses import dataclass, field
from pathlib import Path
from typing import Optional

from harness.main_loop import LoopStatus


@dataclass
class TaskResult:
    """单任务执行结果"""
    task_id: str
    success: bool
    score: float         # 0.0 - 1.0
    turns: int
    total_tokens: int
    elapsed_seconds: float
    cost_usd: float
    error: Optional[str] = None


@dataclass
class BenchmarkReport:
    """完整 Benchmark 报告"""
    harness_version: str
    model: str
    timestamp: float
    task_results: list[TaskResult]
    
    @property
    def success_rate(self) -> float:
        return sum(1 for r in self.task_results if r.success) / len(self.task_results)
    
    @property
    def avg_score(self) -> float:
        return statistics.mean(r.score for r in self.task_results)
    
    @property
    def avg_turns(self) -> float:
        return statistics.mean(r.turns for r in self.task_results)
    
    @property
    def total_cost_usd(self) -> float:
        return sum(r.cost_usd for r in self.task_results)
    
    def to_text(self) -> str:
        lines = [
            f"🏆 Benchmark 报告",
            f"{'='*60}",
            f"Harness 版本: {self.harness_version}",
            f"模型: {self.model}",
            f"评估任务数: {len(self.task_results)}",
            f"",
            f"📊 汇总指标",
            f"  任务完成率: {self.success_rate:.1%}",
            f"  平均得分: {self.avg_score:.2f}/1.0",
            f"  平均轮次: {self.avg_turns:.1f}",
            f"  总成本: ${self.total_cost_usd:.4f} USD",
            f"",
            f"📋 逐任务结果",
        ]
        
        for result in self.task_results:
            status = "✅" if result.success else "❌"
            lines.append(
                f"  {status} [{result.task_id}] "
                f"score={result.score:.2f} "
                f"turns={result.turns} "
                f"${result.cost_usd:.4f} "
                f"{result.elapsed_seconds:.1f}s"
            )
            if result.error:
                lines.append(f"     错误: {result.error[:80]}")
        
        return "\n".join(lines)


class BenchmarkRunner:
    """Benchmark 运行器"""
    
    def __init__(self, harness_factory, version: str = "dev"):
        self.harness_factory = harness_factory
        self.version = version
    
    async def run(
        self,
        tasks: list,
        parallel: bool = False,
    ) -> BenchmarkReport:
        """运行所有评估任务"""
        print(f"开始 Benchmark，共 {len(tasks)} 个任务...")
        
        if parallel:
            results = await asyncio.gather(*[self._run_task(task) for task in tasks])
        else:
            results = []
            for i, task in enumerate(tasks):
                print(f"[{i+1}/{len(tasks)}] 运行: {task.id}")
                result = await self._run_task(task)
                results.append(result)
                print(f"  → {'✅' if result.success else '❌'} score={result.score:.2f}")
        
        return BenchmarkReport(
            harness_version=self.version,
            model="",  # 从 harness 获取
            timestamp=time.time(),
            task_results=list(results),
        )
    
    async def _run_task(self, task) -> TaskResult:
        """运行单个任务"""
        # 环境准备
        if task.setup:
            task.setup()
        
        start = time.time()
        error = None
        score = 0.0
        
        try:
            harness = self.harness_factory(max_turns=task.max_turns)
            
            loop_result = await asyncio.wait_for(
                harness.run(task.description),
                timeout=task.timeout,
            )
            
            # 计算得分
            if task.evaluator and loop_result.status == LoopStatus.COMPLETED:
                score = task.evaluator(loop_result)
            elif loop_result.status == LoopStatus.COMPLETED:
                score = 1.0
            else:
                score = 0.0
            
            success = score >= 0.5
            turns = loop_result.stats.turns
            tokens = loop_result.stats.total_tokens
            
        except asyncio.TimeoutError:
            error = f"任务超时（>{task.timeout}s）"
            turns, tokens = 0, 0
            success = False
        except Exception as e:
            error = str(e)
            turns, tokens = 0, 0
            success = False
        
        finally:
            if task.teardown:
                task.teardown()
        
        return TaskResult(
            task_id=task.id,
            success=success,
            score=score,
            turns=turns,
            total_tokens=tokens,
            elapsed_seconds=time.time() - start,
            cost_usd=0.0,  # 从 cost_tracker 获取
            error=error,
        )
```

## 4. 运行 Benchmark

```bash
# 运行完整 Benchmark
python -m benchmark.run --model claude-opus-4-5 --tasks all

# 只运行特定类别
python -m benchmark.run --category coding --parallel

# 对比两个版本
python -m benchmark.compare reports/v1.json reports/v2.json
```

## 5. 持续集成集成

```yaml
# .github/workflows/benchmark.yml

name: Harness Benchmark

on:
  pull_request:
    branches: [main]

jobs:
  benchmark:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Run Benchmark
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: |
          python -m benchmark.run --output results/pr_${{ github.event.number }}.json
      
      - name: Compare with baseline
        run: |
          python -m benchmark.compare \
            results/baseline.json \
            results/pr_${{ github.event.number }}.json \
            --fail-if-regression 0.05
```

## 📝 本节小结

| 组件 | 职责 |
|---|---|
| `BenchmarkTask` | 定义评估任务（描述、环境、评估函数） |
| `BenchmarkRunner` | 批量运行任务，收集结果 |
| `BenchmarkReport` | 汇总指标和逐任务明细 |
| `Evaluator` | 自动验证任务完成质量（0-1分） |
| CI 集成 | 防止代码变更导致性能回退 |
